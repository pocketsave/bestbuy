Subsystem_2 is being developed with Spring MVC in conjunction with Spring Webflow.

Initial development tools:

1. apache tomcat 8.0.30
2. spring framework 4.2.2 
3. spring webflow 2.4.2
4. eclipse-luna
5. jdk 1.7
6. tiles 2.2.2 (Altough tiles 3.0.5 is available it is not supporting above configuration. Hence forced to use 2.2.2)


Instructions to project setup:

1. Get the project from git 
2. From eclipse, import project as a maven project.
